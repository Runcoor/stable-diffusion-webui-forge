# Model download list for docker/download-models.sh
#
# Format:  <subdir>|<filename>|<url>[|<sha256>]
# Comments start with #. Blank lines are ignored. Whitespace around fields trimmed.
#
# Subdir is relative to MODELS_DIR (default /data/x/models).
# Common subdirs: Stable-diffusion, VAE, Lora, ESRGAN, ControlNet, embeddings, text_encoder
#
# Auth tokens (export before running the script):
#   HF_TOKEN=hf_xxxxx       Required for gated HuggingFace models (e.g. SDXL Base 1.0)
#                            Get one at https://huggingface.co/settings/tokens
#   CIVITAI_TOKEN=xxxxx     Required for most CivitAI models
#                            Get one at https://civitai.com/user/account
#
# Uncomment the lines below for the models you actually want, then run:
#   ./docker/download-models.sh
